# Factory Simulation (Симуляция производственного цеха)

Дискретно-событийная модель работы цеха с умной балансировкой нагрузки. Программа симулирует процесс обработки изделий на станках, маршрутизируя их по наименее загруженным узлам.

## Особенности
- Написано на современном **C++20**.
- Использование **Priority Queue** для обработки событий (`start`, `finish`, `wait`, `ready`, `stop`).
- Надежная обработка ошибок и строгая валидация входных данных (вывод сломанной строки и `exit(1)`).
- Архитектура с соблюдением принципов SOLID (отделен парсинг, логика событий и тестирование).
- Полное покрытие граничных случаев **Google Test**.

## Сборка проекта

Проект использует CMake. Для сборки потребуется компилятор с поддержкой C++20.

```bash
# Клонируем репозиторий
git clone https://github.com/S1r0pchik/Telecom.git
cd Telecom

# Создаем папку для сборки и собираем проект
mkdir build && cd build
cmake ..
cmake --build .
```

# Запуск программы
Программа принимает один аргумент - путь к входному файлу.

```bash
# Находясь в папке build, запустите:
./Telecom ../in.txt
```
